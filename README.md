# xl-stats-func

A javascript implemenation of the statistical functions of Excel.

## Installation

  ```ssh
  $ npm i xl-stats-func
  ```
## Get Started
  ```js

  const {avg, sum} = require("xl-stats-func")
  
  sum(94, 87, 100, 90, 87) // returns 458
  avg(94, 87, 100, 90, 87) // returns 91.6

  ```
